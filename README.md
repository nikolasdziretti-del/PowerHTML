# PowerHTML

**An HTML presentation editor that builds HTML presentations with working HTML interactives.**
One file. Fully offline. No install, no account.

🔗 **Live demo:** https://nikolasdziretti-del.github.io/PowerHTML/

---

## What it is

PowerHTML is a presentation editor that is itself a single `.html` file. Open it in any browser — that *is* the editor. Build your slides, hit **Save**, and you get a second standalone `.html` file that plays your presentation anywhere: from a USB stick, with no internet, no install, and no account.

## Features

- **Interactive video** (offline, H5P-style): pause + notes, single- and multiple-choice questions, numeric answers with tolerance, fill-in-the-blanks, branching (each answer jumps to its own timecode), and a score screen at the end.
- **Built-in video editor** — trim, cut segments, change speed, bleep-out audio.
- **Physics simulations, charts from CSV, QR codes, and formulas** (KaTeX).
- **Presenter view, draw-over-slides, timers, name wheel, dice, confetti.**
- Themes, slide transitions, object animations, and reusable templates.

## How to use

1. Open `PowerHTML_EN_v1.0E.html` in any modern browser (Chrome, Edge, or Firefox).
2. Build your slides — add text, media, questions, simulations.
3. Click **Save**. You get a standalone show file you can share or open anywhere, online or offline.

## Why

No internet dependency, nothing uploaded to a cloud, and the file won't rot when some service shuts down or changes its pricing. Built for teachers, but useful for anyone who wants presentations that aren't twenty bullet-point slides — lecturers, tutors, speakers.

## Honestly

I'm a physics teacher, not a programmer. This was vibecoded — I described what I wanted to an AI, tested it obsessively, and iterated over several months. It works, but it's a large single file, so bug reports are very welcome.
